# To-Do List App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

This project helps in understanding core JavaScript concepts like arrays, objects, CRUD operations, DOM manipulation, and local storage.

---

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Persistent storage using localStorage
- Responsive UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## JavaScript Concepts Learned

- Arrays
- Objects
- CRUD Operations
- Event Listeners
- DOM Manipulation
- forEach()
- map()
- filter()
- Local Storage
- JSON.stringify()
- JSON.parse()
- Dynamic Element Creation

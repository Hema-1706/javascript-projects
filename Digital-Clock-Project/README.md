# Digital Clock

A live digital clock built using HTML, CSS, and JavaScript.

## Features

- Real-time clock
- Hours, minutes, seconds
- AM/PM format
- Modern UI
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Practiced

- Date Object
- Functions
- DOM Manipulation
- Timers
- setInterval()
- String Formatting

## How to Run

1. Download or clone repository
2. Open folder
3. Run `index.html`

# Notes App

A responsive Notes Application built using HTML, CSS, and JavaScript.

This project focuses on CRUD operations, dynamic DOM manipulation, local storage handling, and state management concepts in JavaScript.

---

## Features

- Add notes
- Edit notes
- Delete notes
- Save notes permanently using localStorage
- Responsive card-based UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## JavaScript Concepts Learned

- Arrays & Objects
- CRUD Operations
- DOM Manipulation
- Dynamic Element Creation
- Event Listeners
- localStorage
- JSON.stringify()
- JSON.parse()
- Array Methods
  - forEach()
  - filter()
  - find()
- Date & Time Handling
- State Management Basics

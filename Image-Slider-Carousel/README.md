# Image Slider / Carousel

A responsive Image Slider Application built using HTML, CSS, and JavaScript.

This project focuses on DOM manipulation, intervals, image indexing, and automatic UI updates.

---

## Features

- Next/Previous navigation
- Auto image slideshow
- Responsive design
- Smooth transitions

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## JavaScript Concepts Learned

- Arrays
- DOM Manipulation
- Image Indexing
- setInterval()
- Dynamic UI Rendering
- Event Handling
- Conditional Logic

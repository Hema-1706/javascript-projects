# GitHub Finder

A GitHub profile finder built using HTML, CSS, and JavaScript.

## Features

- Search GitHub users
- View profile image
- View followers and repositories
- Open GitHub profile
- Error handling

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub API

## Concepts Learned

- Fetch API
- Async/Await
- Dynamic HTML Rendering
- JSON Handling
- Error Handling

## How to Run

1. Open project folder
2. Run with Live Server
3. Search GitHub username

# Currency Converter App

A simple currency converter web application built using HTML, CSS, and JavaScript.

## Features

- Convert currencies in real time
- Swap currencies
- Uses live exchange rates
- Responsive UI
- Error handling

## Technologies Used

- HTML
- CSS
- JavaScript
- Exchange Rate API

## Concepts Learned

- DOM Manipulation
- Event Listeners
- Async/Await
- Fetch API
- JSON Handling
- Dynamic Dropdown Creation
- Error Handling

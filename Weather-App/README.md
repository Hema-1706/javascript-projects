# Weather App

A simple weather application built using HTML, CSS, and JavaScript.

## Features

- Search weather by city
- Real-time weather data
- Temperature
- Humidity
- Wind speed
- Error handling

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

## Concepts Learned

- APIs
- Fetch API
- Async/Await
- JSON
- Error Handling
- DOM Manipulation

## How to Run

1. Clone repository
2. Add your API key
3. Open index.html

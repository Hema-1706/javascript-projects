# Counter App

A simple and modern Counter App built using HTML, CSS, and JavaScript.

## Features

- Increase counter
- Decrease counter
- Reset counter
- Dynamic color updates
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Practiced

- State Management
- Event Listeners
- DOM Manipulation
- Functions
- Conditions

## How to Run

1. Clone or download repository
2. Open folder
3. Run `index.html`

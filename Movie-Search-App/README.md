# Movie Search App

A movie search application built using HTML, CSS, and JavaScript.

## Features

- Search movies
- Dynamic movie cards
- Movie posters
- Error handling
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- OMDb API

## Concepts Learned

- Fetch API
- Async/Await
- Dynamic Rendering
- Arrays and Loops
- DOM Manipulation
- Error Handling

## How to Run

1. Add API key
2. Open with Live Server
3. Search movies

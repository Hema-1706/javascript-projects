# Calculator App

A modern calculator built using HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations
- Clear button
- Delete button
- Decimal support
- Responsive UI
- Modern design

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Practiced

- DOM Manipulation
- Event Listeners
- Functions
- Conditions
- Loops
- String Methods

## How to Run

1. Download or clone repository
2. Open folder
3. Run `index.html`
